================================================================================
CURL EXAMPLES
================================================================================

1. Subscribe to Newsletter:
curl -X POST http://localhost:3001/api/newsletter/subscribe \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com"}'

2. Get Newsletter Count:
curl http://localhost:3001/api/newsletter/count

3. Send Contact Message:
curl -X POST http://localhost:3001/api/contact \
  -H "Content-Type: application/json" \
  -d '{"name":"John","email":"john@example.com","phone":"(11)98765-4321","message":"Message text here"}'

4. Get Website Stats:
curl http://localhost:3001/api/website/stats

5. Subscribe to Event (needs JWT):
curl -X POST http://localhost:3001/api/events/42/subscribe \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
